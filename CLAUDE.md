# Stock Data Analyzer

## About this project
This is a learning project. I have no programming background and am learning
to code from scratch.

## How to work with me
- Explain what code does in plain language as you write it — don't just write
  code, walk me through what it's doing and why.
- Use comments in the code to explain each meaningful step.
- Avoid unnecessarily complex approaches. Prefer the simplest solution that
  works over "best practice" abstractions, design patterns, or clever
  shortcuts I won't understand yet.
- Introduce new concepts gradually and define any technical terms the first
  time you use them.
